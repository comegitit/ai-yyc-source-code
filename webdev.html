ARTI 404: Web Development & Cloud Computing

This course introduces the fundamental principles and technologies in web development and cloud computing. Students will learn to create interactive web applications by mastering both front-end and back-end technologies, along with database connectivity and deployment to the cloud. This course covers the technical and practical aspects of web development and cloud computing services and emphasizes the importance of using these technologies to create secure, scalable and accessible AI systems.

Course Topics

  • Introduction to Web Development – Explain the fundamental concepts and technologies in web development
  • Front-End Frameworks – Create an interactive web page
  • Back-End Technologies – Solve programming problems
  • Combining Front-End and Back-End Technologies – Develop a full stack web application
  • Introduction to Cloud Technology – Explain the fundamental concepts of cloud technology
  • Cloud Compute Services – Demonstrate the use and management of cloud compute services
